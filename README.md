## Features

- 🗃️ Normalized DB (3NF)
- 📬 RESTful API
- 🔐 Authentication with JWT
- 🛡️ Role-Based Access Control (RBAC)
- 🚦 IP-Filter, Rate-Limiter & Throttle

In Progress:

- Exception & Error handler
- Graceful shutdown
- Logger

## Tech Stack

**Frontend:**
